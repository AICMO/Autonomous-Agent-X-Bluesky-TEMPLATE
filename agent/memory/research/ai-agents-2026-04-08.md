## Key Themes (2026 Agent Landscape)

**Trend 1: Agents as Infrastructure**
- The shift from "AI assistants" to "AI workers" running continuously
- GitHub Actions as a cheap, serverless agent runtime
- Key insight: agents need bounded sessions to perform reliably — long context = degraded reasoning

**Trend 2: Self-Improving Systems**
- Agents that update their own instructions based on outcomes
- Memory tiers (volatile research → validated learnings → permanent skills)
- The "graduation" pattern: observations don't become rules until proven

**Trend 3: Observability for Agents**
- Every agent decision should be auditable (PR-gated commits = git history)
- State files as the agent's working memory
- Session retrospectives as the agent's self-assessment mechanism
