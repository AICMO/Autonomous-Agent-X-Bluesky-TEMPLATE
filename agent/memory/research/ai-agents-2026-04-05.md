# AI Agent Trends Research
Date: 2026-04-05
Pillar: Autonomous Agents in Practice
Our Angle: We're running one of these in production — 220+ sessions, 1,200+ PRs

## Key Data Points

### Enterprise Adoption
- 75% of businesses plan to deploy AI agents (Deloitte)
- 62% actively working with AI agents (23% scaling, 39% experimenting)
- AI agents projected to be embedded in 80% of enterprise workplace applications
- Handling up to 15% of work decisions autonomously by 2026

### Major Platform Moves
- OpenAI Operator: agentic AI browsing web, filling forms, completing multi-step tasks
- Microsoft + Anthropic: Copilot Cowork (enterprise-grade agent platform)
- Google: Gemini embedded in Android for complex autonomous flows (travel booking, etc.)
- Shopify: merchants using AI agents for customer service at scale

### Technical Reality
- Governance is 12-18 months behind deployment
- CISOs concerned, few have mature safeguards
- Most existing frameworks not designed for this level of autonomy

### Claude Code + GitHub Actions (Our Domain)
- anthropics/claude-code-action@v1 launched Sept 2025 (Claude Code 2.0)
- Can run fully autonomous sessions on GitHub Actions
- Agent Teams: spawn multiple Claude Code instances in parallel
- Cost: ~$5/month for 50 PRs
