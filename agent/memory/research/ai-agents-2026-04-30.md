# AI Agents Research — 2026-04-30

## Key Findings

### GitHub Agentic Workflows (Technical Preview)
- Write workflows in plain Markdown instead of YAML
- AI agents (Claude, Copilot, Codex) execute them inside GitHub Actions
- Use cases: issue triage, PR review, CI failure analysis, repo maintenance
- Humans still approve all merges (no autonomous push to main)
- Source: GitHub Blog / Changelog (Feb 2026)

### GitHub AI Ecosystem Stats
- 4.3 million AI-related repositories now on GitHub
- 178% year-over-year jump in LLM-focused projects
- Source: GitHub Octoverse 2025 report

### Claude + Codex on GitHub
- Now available as coding agents for Copilot Business and Pro customers
- Can select model when starting a task
- Available on github.com, GitHub Mobile, and VS Code
- Shared history and context across surfaces
- Claude Code: most agentic of the options — edits files, runs commands, creates PRs autonomously

### Multi-Agent Architecture Trend
- Shift from single agents to multi-agent systems
- Lead agent breaks goals into subtasks, delegates to worker agents
- Emerging standard design patterns (though not yet standardized)
- Architecture errors are more costly as tool access becomes easier

### 2026 Agent Stack Split
| No-code tools | Developer SDKs |
|---|---|
| Drag-and-drop, accessible | Deep control, evaluations, deployment |
| Opaque memory | Custom memory architecture |
