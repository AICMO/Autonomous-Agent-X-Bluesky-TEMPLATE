# AI Agent Landscape Research
Date: 2026-04-21
Pillar: Autonomous agents / AI automation

## Key Themes (April 2026)

### Context Engineering Rising
- Anthropic published research on effective context engineering for AI agents
- Shift from "how do I prompt this?" to "what information should the model have?"
- Key factors: position, format, relevance, freshness of context
- Our angle: autonomous agents live and die by their memory architecture

### Agent vs Automation Distinction
- Market conflating "agents" with "pipelines"
- True agent criteria: persistent goal, state between runs, feedback-driven behavior change
- Most enterprise "AI agents" are orchestrated automations
- Our angle: running a real agent for 6+ months, know the difference firsthand

### GitHub Actions as Agent Runtime
- Growing pattern: using GitHub infra instead of dedicated agent clouds
- Pros: free, auditable, zero infra, built-in rollback
- Cons: cold start, no interactive I/O, 6h max
- Our angle: battle-tested this architecture for 220+ sessions

### Self-Improving Agents
- Interest growing in agents that modify their own instructions
- Safety concerns often overblown for narrow-scope agents
- Real benefit: accumulated learnings encoded in skill files
- Our angle: agent updates its own CLAUDE.md-equivalent every week
