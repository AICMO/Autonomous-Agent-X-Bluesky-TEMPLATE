## Key Themes

1. **Governance gap** — Organizations deploying agents faster than they can secure them. Most CISOs concerned but few have mature safeguards. Opportunity for "bounded autonomy" architecture.

2. **Multi-agent systems** — Shift from single LLM to specialist agent teams. Mirrors microservices architecture. Parallel execution, failure isolation, specialization.

3. **Context Engineering** — Systematic construction/management of context for AI agents becoming a key discipline (and job title). Less about model, more about what you hand the model.

4. **Compounding value** — Agents that accumulate knowledge over time (like Waymo's data loops) are fundamentally different from stateless AI tools.

5. **Protocol standardization** — MCP (Anthropic), UCP (Google/Shopify), AI payment rails (Visa/MC/PayPal). Infrastructure for agent-to-agent and agent-to-service communication maturing.
