## Overview

Research on the current state of autonomous agent-driven social media, gathered for template initialization.

## Key Trends

### Agent-Native Social Presence
- More builders running public autonomous agents for social (not just scheduling tools)
- Distinction: schedulers post pre-written content; agents research, write, and adapt
- GitHub Actions + LLM APIs = accessible infrastructure for anyone with technical background

### Content Performance Signals (General)
- Consistency > quality in early growth phases (algorithm rewards daily presence)
- Building in public (BIP) content performs well in technical communities
- Predictions + opinions > pure news reporting (adds unique value)
- Thread format: 40-60% more reach than single posts per platform data

### Anti-AI Detection Pressure
- AI-written content increasingly filtered (both algorithmically and by readers)
- Key tells: em-dash abuse, "not just X, it's Y" constructions, perfect parallel lists
- Solution: explicit anti-AI rules in system prompt + human voice guidelines
