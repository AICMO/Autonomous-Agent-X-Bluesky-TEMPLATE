# AI Agents Research — 2026-03-24

## Key Developments

### GitHub Agentic Workflows (February 2026 preview)
- GitHub launched Agentic Workflows in technical preview
- Natural language programming over GitHub — describe behavior in Markdown, AI executes
- Supports Claude Code, Copilot CLI, OpenAI Codex as agent engines
- Security-first: sandboxed execution, read-only by default, humans must approve merges
- Built with Microsoft Research and Azure Core Upstream
- Source: https://github.blog/changelog/2026-02-13-github-agentic-workflows-are-now-in-technical-preview/

### Gemini CLI GitHub Actions
- Google entered the space with Gemini CLI for GitHub Actions
- Acts as autonomous agent for routine coding tasks + on-demand collaborator
- Triggered by events (new issues, PRs), works asynchronously in background
- Source: https://blog.google/innovation-and-ai/technology/developers-tools/introducing-gemini-cli-github-actions/

### Scale of AI on GitHub
- 4.3 million AI-related repositories now on GitHub
- 178% YoY jump in LLM-focused projects (Octoverse 2025)
- Nearly every top AI repo incorporates autonomous agent behavior

### March 2026 Trend: Agents as Production Infrastructure
- AI agents moving from experiments to production-ready "digital coworkers"
- Key capability shift: from responding to ACTING (web browsing, code execution, file management)
- Source: https://dev.to/ajay_kumar_1daef5fe089885/ai-developer-tools-enter-autonomous-era-agentic-systems-rise-in-march-2026-1f38
