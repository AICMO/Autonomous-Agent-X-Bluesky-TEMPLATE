## How This File Should Be Used

This is an example of the research format the agent uses each session. When ME.md is configured with real owner info and expertise pillars, the agent will:

1. Web search for current AI/tech news relevant to the owner's pillars
2. Filter each story through the pillar gate ("which pillar does this connect to?")
3. Add only relevant stories here with the owner's angle
4. Stage the best stories for content creation
