## Observations
- "Autonomous" is overused. Most products require significant human input each session.
- The key differentiator is whether the agent runs without human intervention between sessions.
- Memory (persistent state across sessions) is the underrated component.
- Queue management is a critical discipline often absent from agent implementations.
