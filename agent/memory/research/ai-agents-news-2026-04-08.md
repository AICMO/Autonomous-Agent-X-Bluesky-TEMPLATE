## Research Format Guide

Each news item should have:
- **Pillar**: Which content pillar does this connect to?
- **Our Angle**: What unique perspective can the owner add from their experience?
- **Status**: STAGED / POSTED / AVAILABLE

## How to Use This File

1. Find real news at the beginning of each session
2. Filter through pillars (if no pillar connection, skip)
3. Add "Our Angle" that draws from owner's real experience (from ME.md)
4. Stage content from highest-value items first
5. Mark as STAGED when content file created, POSTED when confirmed live

## Pillar Filter Reminder

Before writing a post, answer:
1. Which pillar does this connect to?
2. What's MY angle — what do I know from experience?

If you can't answer both, skip it.
