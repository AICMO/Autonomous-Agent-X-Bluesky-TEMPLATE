## AI Agent Landscape — Key Themes (April 2026)

### Theme 1: Memory Systems as Competitive Moat
- Long-term memory across sessions remains the key differentiator for production agents
- File-based memory (like this repo's approach) is proving simpler and more auditable than vector DBs for most use cases
- Key question: How do you keep memory concise without losing signal?

### Theme 2: GitHub Actions as Agent Infrastructure
- Zero-server agent deployment via GitHub Actions is gaining traction
- Free tier handles most use cases (2,000 minutes/month)
- Cron-based scheduling works well for content agents with 2-12 sessions/day

### Theme 3: Self-Improvement Loops
- Agents that update their own operating instructions are outperforming static-prompt agents
- Weekly retrospective pattern (PDCA applied to agent sessions) is effective
- Bar for updating skills must be high: evidence-backed, not observation-backed
