## Key Data Points

### GitHub Scale
- AI agent PRs surged from ~4M (Sept 2025) to 17M+ (March 2026) — 4x in 6 months
- GitHub processed 275M AI agent commits per week as of April 2026
- Single week in April 2026: 2.1 billion compute minutes (causing infrastructure strain)
- 14 billion AI-agent commits projected for full year 2026

### GitHub Agentic Workflows (Technical Preview, Feb 2026)
- Write workflows in plain Markdown instead of YAML
- Native access to repos, issues, PRs, Actions, security via GitHub MCP Server
- Additional tools: browser automation, web search, custom MCPs
- New: `pre-agent-steps` frontmatter field for setup before AI agent runs
- New: OpenCode engine option (joins Copilot, Claude, Codex as first-class options)
- Security fix: working tree scanned for planted executables before each run

### Developer Adoption
- 68% of developers now rely on AI for at least half of their production code (Stack Overflow 2026 survey)
- GitHub Copilot Workspace resolved 78% of GitHub issues autonomously in 2026 benchmark
- Cloudflare uses 7-agent coordinated code review: security, performance, quality, docs, release, compliance + coordinator

### Self-Improving Systems
- Andrej Karpathy open-sourced autoresearch (March 2026): 630-line Python script
  - AI agent modifies training code, runs experiments, evaluates, iterates
  - Ran 700 experiments in 2 days
  - Discovered 20 optimizations improving efficiency by 11%

### Frameworks (2026 ecosystem)
- CrewAI — role-playing autonomous agents, collaborative intelligence
- Microsoft Agent Framework — AutoGen + Semantic Kernel merged, GA Q1 2026
- Factory.ai — agent-native software development platform
