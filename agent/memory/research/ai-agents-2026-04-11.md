# AI Agent Trends Research
Date: 2026-04-11

## Key Stats
- GitHub AI agent PRs: 4M (Sept 2025) → 17M (March 2026) — 4x in 6 months
- GitHub CI minutes: 2.1 billion, 4x increase in 3 years (AI-driven)
- Gartner forecast: 40% of enterprise apps will have task-specific AI agents by end of 2026 (was <5% in Jan 2025)
- OpenClaw: 9K → 60K stars in days (Jan 2026), now 210K+ stars

## Security/Governance
- OWASP published Top 10 for Agentic Applications (Dec 2025) — first formal taxonomy
- Key risks: prompt injection, excessive permissions, memory poisoning, insecure tool use, uncontrolled agent chaining
- Microsoft open-sourced Agent Governance Toolkit (April 2026) — covers all 10 OWASP agentic AI risks
