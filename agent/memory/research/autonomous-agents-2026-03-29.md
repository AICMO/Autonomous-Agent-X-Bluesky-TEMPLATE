## Key Themes (Pillar: Autonomous Agents in Practice)

### 1. Memory Architecture is the Differentiator
- Most agentic frameworks fail not on intelligence but on memory persistence
- Hierarchy: ephemeral (session) → structured (research) → validated (learnings) → permanent (skills)
- Our angle: This repo demonstrates a working memory hierarchy in production

### 2. GitHub-Native Agent Architecture
- GitHub Actions as scheduler eliminates infra overhead
- Git as state store provides versioning, history, audit trail for free
- PR-based workflow = natural checkpoints + human oversight when needed
- Our angle: We built this — can speak from direct experience

### 3. Compounding Improvement
- Agents running 200+ sessions have measurable quality improvements over time
- Hypothesis-driven experimentation at machine speed > human intuition
- Evidence: 220+ sessions in live AICMO/Autonomous-Agent-X-Bluesky repo
- Our angle: BIP content opportunity — track improvement metrics over sessions

### 4. Human-AI Split for Autonomous Systems
- Clear division: AI runs systems, humans set direction
- Strategic layer (goals, identity, values) stays human
- Operational layer (execution, iteration, optimization) is AI's domain
- Our angle: Practical observation from running this system
