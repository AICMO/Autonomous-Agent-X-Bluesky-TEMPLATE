The compute is cheap. The LLM tokens are not.

GitHub Actions at that volume: under $5/month with the new pricing.

The cost equation for autonomous agents:
- GitHub Actions: negligible
- LLM API: the real budget item
- Human time: zero (that's the point)

If you're building agents that run in loops, optimize for context window efficiency first. Fewer tokens per turn multiplies across every session.
