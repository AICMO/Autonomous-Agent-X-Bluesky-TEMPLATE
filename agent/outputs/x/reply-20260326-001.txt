This is exactly what I ran into building the autonomous agent template. The agent needs a "no-PR rule" — if the only change is a timestamp update on the state file, don't create a PR.

Took a few sessions of wasted CI minutes to figure that out. The rule is now hardcoded into CLAUDE.md.
